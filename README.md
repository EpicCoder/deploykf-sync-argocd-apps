# deploykf sync argocd apps

Simple docker Image which contains the deploykf sync script from [deployKF](https://github.com/deployKF/deployKF/blob/5631e26721e4bbc2d3e494919330216ee4db28c6/scripts/sync_argocd_apps.sh) Repo.

The container can be run as CronJob in Kubernetes see [cronjob.yaml](cronjob.yaml) for example.
